# Semantic Change in Nineteenth- and Twentieth-Century French Literary Prose

This repository contains the code developed for my Master's dissertation in Computational Linguistics at Sorbonne Nouvelle.

## Thesis

The project investigates whether the statistical laws of semantic change proposed by Hamilton, Leskovec and Jurafsky (2016) hold in a large corpus of 19th- and 20th-century French literary prose. It first reproduces the original static embedding methodology before evaluating its robustness under different preprocessing and modelling conditions. The second part of the project explores contextual embedding models to examine whether semantic change patterns remain consistent across modelling paradigms.
